# Isometric World

A small web application that displays an isometric world made of different tile types.

## Tile types

- **Grass** – green terrain
- **Water** – blue
- **Sand** – tan
- **Stone** – gray
- **Forest** – dark green

The map is generated with a mix of these types. The world is drawn in classic 2:1 isometric projection.

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
# or
python3 -m http.server 8080
```

Then open the URL shown (e.g. http://localhost:3000 or http://localhost:8080).
